Files contained within…

Driver.java
Contains a main method that instantiates and uses MapDataDrawer to do the work of finding and drawing mountain paths.

MapDataDrawer.java
an object that maintains a 2D array of mountain data and methods to operate on it.

MountainPaths.java
also contains a main method…solving the same problem(s) just with static methods in a single class.

DrawingPanel.java
a class from Reges and Stepp that makes drawing onto a JPanel inside a JFrame easy and straightforward.  Also contains nice UI utilities to save the image created, zoom and and out, etc.
